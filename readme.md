# Astro Dex Back End

## Installation

1. Clone the repository:
``` bash
git clone https://github.com/eynomr/ASTRODEX-BACKEND.git
```
2. Install the dependencies:
```bash
pip install -r requirements.txt
```
3. Running the server
``` bash
flask run
```
